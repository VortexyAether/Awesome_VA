# Paper Writing Tools

Tools and workflows for academic writing, LaTeX, citation management, figures, reviews, and proposals.

## Overleaf

- Link: https://www.overleaf.com/
- Type: Tool / online LaTeX editor
- Keywords: LaTeX, collaboration, academic writing, paper drafting
- One-line summary: A collaborative online LaTeX editor for writing papers, reports, theses, and technical documents.
- Why it matters: LaTeX remains the standard for many CFD/SciML papers, and Overleaf reduces friction around collaboration, templates, versioning, and reviewer-ready manuscript preparation.
- Possible use: Maintain paper drafts, thesis chapters, lab reports, and reproducible figure/table workflows connected to Git or local research notes.
- Maturity: production-ready
- Priority: High

## Inkscape

- Link: https://inkscape.org/
- Type: Tool / vector graphics editor
- Keywords: SVG, figures, diagrams, publication graphics, open source
- One-line summary: An open-source vector graphics editor for creating and editing SVG-based scientific figures and diagrams.
- Why it matters: Clear diagrams are essential for explaining CFD workflows, neural operator architectures, geometry pipelines, and result interpretation in papers and presentations.
- Possible use: Create methodology diagrams, annotate exported plots, polish schematic figures, and edit vector assets for LaTeX/Overleaf manuscripts.
- Maturity: production-ready
- Priority: High
