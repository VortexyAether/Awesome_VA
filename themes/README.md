# Themes

Long-running technical threads promoted from daily radar.

Use this directory when a signal appears repeatedly across days and deserves a stable synthesis page.

Recommended theme pages:

- `validation-gated-surrogates.md` — no-harm ML, fallback contracts, solver-aware validation, audit logs.
- `geometry-family-generalization.md` — train/test splits by unseen geometry family, CAD/BRep-aware benchmarks.
- `cad-native-engineering-ai.md` — BRep, feature graphs, executable CAD, MCP/CAD automation.
- `sparse-sensor-field-reconstruction.md` — sensor-to-field recovery, Voronoi/spectral/diffusion reconstruction, experimental data assimilation.
- `agentic-digital-twins.md` — agents proposing changes under validation gates, digital twins with logs and rollback.

Each theme page should contain:

```markdown
# Theme Name

## Thesis
The reusable idea.

## Why it matters
Engineering consequence.

## Key references
- Link + one-line role.

## Evaluation checklist
What must be checked before trusting this idea.

## VA experiment hooks
Concrete experiments, metrics, or demos VA could build.
```
