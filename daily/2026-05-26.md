## Added Resources

### Thermal-fluid datasets, CFD surrogates & solver-assist ML

- [Open Multimodal Datasets and Open-Source Software for Data-Driven Modeling of Multiphase Transport and Thermal Systems](https://arxiv.org/abs/2605.23037) — NED3 Lab의 boiling image, acoustic/thermal measurement, high-speed video, IR thermography, thermal-resistance, CFD field, design-file datasets와 BubbleID/SeqReg/CFDTwin/IRISApp/FlowLab류 도구를 한 생태계로 정리합니다. thermal-fluid AI에서 데이터/decoder/baseline/tooling을 같이 보는 점이 고신호입니다. `#thermal-fluid` `#dataset` `#multiphase` `#open-source`
- [Physics-informed convolutional neural networks for fluid flow through porous media](https://arxiv.org/abs/2605.20250) — complex pore geometry에서 velocity field를 예측하되 incompressibility, no-flow-in-solid, periodicity, tortuosity loss를 함께 쓰고, 예측장을 Lattice-Boltzmann 초기조건으로 넣어 solver iteration을 줄입니다. surrogate를 solver 대체가 아니라 warm-start로 쓰는 현실적인 패턴입니다. `#porous-media` `#solver-assist` `#physics-informed` `#LBM`
- [Conditional Neural Field based Reduced Order Model for Dynamic Ditching Load Prediction](https://arxiv.org/abs/2605.21499) — aircraft ditching load를 conditional neural field + latent LSTM으로 예측하며, heterogeneous spatial discretization에서도 reconstruct 가능한 점이 핵심입니다. remeshing이 잦은 CAD/CFD design loop의 fixed-grid surrogate 한계를 피하는 reference입니다. `#ROM` `#neural-field` `#CFD-loads` `#mesh-flexibility`
- [ShapeBench: A Scalable Benchmark and Diagnostic Suite for Standardized Evaluation in Aerodynamic Shape Optimization](https://arxiv.org/abs/2605.20763) — 103개 aerodynamic shape optimization task, validated surrogate, 가능한 경우 high-fidelity CFD final verification을 묶은 benchmark. surrogate optimum이 실제 CFD에서 살아남는지 보는 protocol이 VA 관심사와 잘 맞습니다. `#aerodynamics` `#shape-optimization` `#benchmark` `#CFD-verification`

### Control, CAD/CAE agents & engineering automation

- [PI-SONet: A Physics-Informed Symplectic Operator Network for Real-Time Optimal Control of Multi-Agent Systems](https://arxiv.org/abs/2605.14332) — parameterized PMP optimal-control family의 solution map을 구조보존 neural operator로 학습해 sub-second inference를 노립니다. thermal/control/flow-control surrogate를 “반복 optimal-control query 가속기”로 보는 데 좋은 anchor입니다. `#optimal-control` `#neural-operator` `#structure-preserving` `#real-time-control`
- [COSMO-Agent: Tool-Augmented Agent for Closed-loop Optimization, Simulation, and Modeling Orchestration](https://arxiv.org/abs/2605.20190) — CAD generation, CAE solving, result parsing, geometry revision을 외부 tool orchestration RL 환경으로 구성합니다. CAD-CAE semantic gap, 즉 simulation feedback을 valid geometric edit로 바꾸는 병목을 정확히 찌릅니다. `#CAD-CAE` `#agentic-engineering` `#closed-loop` `#design-automation`
- [AutoCAD MCP Server](https://github.com/wade20250715/autocad-mcp-server) — AutoCAD/DWG를 MCP로 agent-addressable하게 만들려는 초기 repo. 성숙도와 adoption은 낮지만, FreeCAD/CadQuery뿐 아니라 상용 CAD도 typed control surface로 열리기 시작한다는 workflow signal로만 볼 만합니다. `#MCP` `#AutoCAD` `#CAD-automation` `#early-tool`

## Possible Research Ideas

1. **Thermal-fluid data inventory template:** NED3의 S+TD 기준으로 VA가 모을 수 있는 데이터셋을 `0+1D sensor`, `2+0D IR/image`, `2+1D video`, `3+0D CFD field`, `design file`로 분류하고, 각 항목에 decoder/baseline/metric을 강제한다.
2. **CFD warm-start benchmark:** porous-media 논문의 패턴을 따라 작은 LBM/OpenFOAM case에서 ML-predicted initial field가 residual convergence와 wall-clock을 얼마나 줄이는지 측정한다. 모델 정확도보다 solver iteration 절감률을 KPI로 둔다.
3. **Mesh-change-resistant surrogate test:** conditional neural field ROM 아이디어를 geometry variant / remeshed duct or heat-sink dataset에 적용해 fixed-grid surrogate와 비교한다.
4. **CAD-CAE agent gate:** COSMO-Agent 스타일로 CAD edit → export → mesh smoke test → solver mini-run → KPI parse → next edit를 최소 pipeline으로 만들고, 실패 위치를 provenance log로 남긴다.
