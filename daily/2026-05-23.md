# Daily Research Radar — 2026-05-23

## Highlights

- **오늘의 핵심 신호는 “surrogate를 최적화/제어 루프 안에서 검증하자”입니다.** Weak-form latent dynamics, ShapeBench류 benchmark, thermal digital twin 흐름이 모두 단순 field prediction보다 repeated design/control query 비용을 줄이는 방향으로 모입니다.
- **멀티스케일 물리 emulation은 generative model을 쓰되 비용과 rollout 안정성을 같이 잡으려는 쪽으로 갑니다.** Wavelet Flow Matching은 fine-scale structure 보존과 장기 rollout 안정성 사이의 타협점을 wavelet/latent flow matching으로 찾습니다.
- **engineering agents는 “자연어 → solver 입력 → artifact 검증 → parameter revision” 루프가 더 구체화되고 있습니다.** TO-Agents와 CAD/FreeCAD MCP 도구들은 design intent를 실제 최적화/geometry workflow로 연결하려는 신호입니다.
- **MCP/skill 쪽은 governance와 verification이 중요해지고 있습니다.** Contractual Skills와 CadGate는 agent가 뭘 할 수 있는지뿐 아니라 evidence, output contract, geometry diff, DFM rule 같은 검증 표면을 명시해야 한다는 흐름입니다.

## Added Resources

### SciML, neural operators & PDE optimization

- [Time-Dependent PDE-Constrained Optimization via Weak-Form Latent Dynamics](https://arxiv.org/abs/2605.20639) — high-dimensional time-dependent PDE 최적화에서 반복 forward/sensitivity solve 비용을 줄이기 위해 weak-form latent-space ROM을 쓰는 접근. thermal/control/CFD design loop에서 “빠른 surrogate + gradient optimization”을 만들 때 직접 참고할 만합니다. `#PDE-optimization` `#reduced-order-model` `#control` `#SciML`
- [Wavelet Flow Matching for Multi-Scale Physics Emulation](https://arxiv.org/abs/2605.16573) — PDE 기반 multi-scale system emulation에서 deterministic surrogate의 over-smoothing과 diffusion/flow matching의 비용 문제를 동시에 다룹니다. turbulence, climate, thermal field처럼 fine-scale structure가 중요한 rollout surrogate 후보입니다. `#flow-matching` `#multi-scale-physics` `#generative-surrogate` `#PDE`
- [Symmetry in the Wild: The Role of Equivariance in Neural Fluid Surrogates](https://arxiv.org/abs/2605.18816) — real-world CFD surrogate에서 geometry/mesh scale, architecture, symmetry/equivariance가 실제로 얼마나 도움이 되는지 묻는 paper. “equivariant라서 좋다”가 아니라 large surface/volume mesh에서 실용성이 있는지 보는 방향이라 의미 있습니다. `#CFD-surrogate` `#equivariance` `#mesh` `#aerodynamics`
- [Physics-based Digital Twins for Integrated Thermal Energy Systems Using Active Learning](https://arxiv.org/abs/2605.06756) — 이전에 잡은 항목이지만 오늘 thermal/control 관점에서 다시 중요. Modelica thermal-system simulation, active learning, interpretable surrogate를 묶어 real-time supervisory control을 겨냥합니다. `#thermal` `#digital-twin` `#active-learning` `#control`

### CAD, topology optimization & engineering agents

- [TO-Agents: A Multi-Agent AI Pipeline for Preference-Guided Topology Optimization](https://arxiv.org/abs/2605.21622) — natural-language design intent를 validated solver inputs로 바꾸고, topology optimization 결과를 3D render/VLM judge로 critique한 뒤 parameter를 수정하는 multi-agent loop. CAD/CAE 자동화에서 “예쁜 말”을 solver-controllable objective/constraint로 번역하는 병목을 잘 찌릅니다. `#topology-optimization` `#engineering-agents` `#design-automation` `#VLM`
- [sandraschi/freecad-mcp](https://github.com/sandraschi/freecad-mcp) — FastMCP 3.2 기반 FreeCAD MCP server + webapp이며 설명상 FluidX3D/OpenFOAM CFD extension을 포함합니다. 아직 star 1 수준의 초기 repo지만 CAD agent가 geometry 생성에서 CFD handoff까지 욕심내는 방향의 신호입니다. `#FreeCAD` `#MCP` `#OpenFOAM` `#CAD-CAE`
- [CadGate](https://github.com/vericontext/cadgate) — AI-generated CAD-as-code PR을 CadQuery/Build123d 기준으로 geometric metric diff, min-wall-thickness, DFM rules, 6-view rendered PR preview로 검증하려는 CLI/GitHub Action. agentic CAD에서 생성보다 더 중요한 “변경 검증 gate” 쪽이라 추적 가치가 큽니다. `#CAD-as-code` `#verification` `#CadQuery` `#DFM`
- [Contractual Skills: A GovernSpec Design Framework for Enterprise AI Agents](https://arxiv.org/abs/2605.22634) — skill 파일을 goals, permissions, evidence requirements, output contracts, verification steps, approval points까지 포함하는 readable task contract로 보자는 제안. engineering agent workflow에도 바로 적용 가능합니다: solver run, artifact provenance, human approval, failure handoff를 skill 수준에서 명시하는 방향. `#agent-governance` `#skills` `#MCP` `#verification`

## Best Find of the Day

**CadGate + Contractual Skills 조합**이 오늘의 best workflow signal입니다. 새 surrogate 논문도 좋지만 VA의 CAD/CFD agent workflow에는 “agent가 만든 CAD/solver artifact를 어떻게 믿을 것인가”가 더 큰 병목입니다. CadGate는 geometry diff/DFM/render preview라는 실물 검증 gate를, Contractual Skills는 permission/evidence/output contract라는 운영 contract를 제공합니다. 둘을 합치면 CAD→mesh→CFD loop의 최소 안전장치를 설계하기 좋습니다.
