1. [Domain-Validity-Gated Metamorphic Testing of Scientific ML Surrogates](https://arxiv.org/abs/2606.17529) — SciML surrogate의 oracle problem을 다루기 위해 metamorphic relation을 무작정 적용하지 않고, domain-validity rubric, MR-card executable asset, claim ledger로 admit/reject/execute verdict를 남기는 프레임워크입니다. MeshGraphNets cylinder flow, compressible airfoil, FNO Burgers/heat까지 테스트합니다. **VA에 중요한 이유:** VA가 원하는 CFD/thermal surrogate governance는 “모델이 맞았다”보다 어떤 물리/수치 조건에서 어떤 검증 카드가 유효했는지 기록해야 합니다. **Action: Test** `#SciML-validation` `#CFD` `#metamorphic-testing` `#claim-ledger`
2. [Physics-conforming Latent Twins](https://arxiv.org/abs/2606.15053) — latent surrogate solution operator가 conservation law, invariants, admissibility, dissipation 구조를 보존/소산하도록 constraint-transfer 관점으로 학습하는 방법입니다. **VA에 중요한 이유:** time-dependent PDE surrogate는 interpolation accuracy만으로 부족하고 장시간 rollout에서 물리 결함이 누적되지 않는지 봐야 하므로, latent-space 구조 보존은 thermal/flow digital twin의 신뢰성 기준이 됩니다. **Action: Read** `#latent-twin` `#structure-preserving` `#PDE-surrogate` `#digital-twin`
3. [ReproRepo: Scaling Reproducibility Audits with GitHub Repository Issues](https://arxiv.org/abs/2606.18237) — 1,149개 ML paper/repo pair와 human-raised GitHub issue를 이용해 LLM agent의 reproducibility blocker 탐지 능력을 평가하는 프레임워크입니다. **VA에 중요한 이유:** CFD/CAE paper reproduction도 “agent가 요약했다”가 아니라 repo issue, build failure, missing data, wrong defaults 같은 real blocker를 찾아 artifact로 남기는 평가가 필요합니다. **Action: Watch** `#reproducibility` `#agent-evaluation` `#GitHub` `#research-audit`

### CFD / thermal / hypersonic / hydrodynamics

4. [Accelerating Kinetic Fokker-Planck Simulations via a GPU-Native Deep Neural Network Surrogate](https://arxiv.org/abs/2606.15622) — cubic Fokker–Planck closure 계산을 GPU-native neural surrogate로 particle simulation loop 안에서 대체하고, rarefied internal flow 및 hypersonic cylinder에서 runtime, conservation/stability, entropy-proxy fidelity를 검증합니다. **VA에 중요한 이유:** learned closure를 solver 밖 post-processing이 아니라 online solver loop에 넣는 사례이며, entropy/near-wall audit 같은 검증 항목이 실용적입니다. **Action: Read** `#rarefied-flow` `#hypersonic` `#learned-closure` `#GPU`
5. [Multiscale Hypersonic Boundary Layer Reconstruction via Spectral Binning and Subdomain-wise Conditional Diffusion](https://arxiv.org/abs/2606.15023) — top-wall 관측으로 Mach 6–8 hypersonic Couette near-wall fields를 조건부 diffusion으로 복원하고, bounded binned spectral power loss로 high-wavenumber fidelity를 보존합니다. **VA에 중요한 이유:** hypersonic/thermal boundary layer surrogate는 평균 profile보다 small-scale spectra, wall quantities, uncertainty 구조가 중요하다는 좋은 benchmark lens입니다. **Action: Save** `#hypersonic` `#conditional-diffusion` `#spectral-loss` `#uncertainty`
6. [ShipNet: A Geometric Deep Learning Surrogate for Real-Time Ship Hydrodynamics](https://arxiv.org/abs/2606.15356) — hull point cloud와 speed에서 hull pressure 및 far-field free-surface wave pattern을 예측하는 geometric DL surrogate입니다. geometry-held-out test와 0.15s inference를 보고하지만, 두 parent yacht hull과 inviscid potential-flow data에 제한됩니다. **VA에 중요한 이유:** CAD/geometry→hydrodynamic surrogate의 좋은 응용이지만, geometry-family shift와 viscous/high-fidelity 확장이 핵심 리스크입니다. **Action: Watch** `#hydrodynamics` `#geometry-surrogate` `#ship-design` `#graph-learning`
7. [Online Spectral Deflation for State Constrained Optimal Control Problems](https://arxiv.org/abs/2606.17971) — state-constrained PDE optimal control의 active-set Schur complement solve를 surrogate로 대체하지 않고, reference eigenmodes를 inactive set에 restrict해 Jacobi-CG를 가속합니다. diffusion, convection-diffusion, nonlinear thermal, conjugate heat-transfer에서 55–98% iteration reduction을 보고합니다. **VA에 중요한 이유:** ML surrogate가 아니어도 design/control loop 병목을 줄이는 solver-native acceleration은 강력한 baseline입니다. thermal control/optimization에서 “surrogate vs robust linear algebra reuse” 비교 기준이 됩니다. **Action: Cite** `#optimal-control` `#thermal` `#conjugate-heat-transfer` `#solver-acceleration`
