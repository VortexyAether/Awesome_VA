## Added Resources

### CFD / thermal-fluid surrogate and control

- [Accelerating Bayesian inverse design in computational fluid dynamics using neural operators](https://arxiv.org/abs/2605.26059) — neural operator를 Bayesian inverse-design loop 안에 넣어 sparse flow observations에서 geometry posterior를 추정합니다. CFD surrogate를 “빠른 예측기”가 아니라 uncertainty-aware design inference engine으로 쓰는 방향이라 VA의 aerodynamic/design automation 관심사와 잘 맞습니다. `#CFD` `#inverse-design` `#neural-operator` `#uncertainty`
- [Operator Learning for Reconstructing Flow Fields from Sparse Measurements: a Language Model Approach](https://arxiv.org/abs/2605.23712) — sparse sensor measurements에서 mesh-free 방식으로 flow field를 복원하는 operator-learning 접근입니다. 실험/운영 데이터가 dense field가 아니라 센서 몇 개로 들어오는 현실을 반영하므로 flow control·digital twin에 직접 연결됩니다. `#sparse-sensing` `#flow-reconstruction` `#operator-learning` `#control`
- [Sparse POD Mode Selection and Manifold Dimensionality Reduction with Neural Networks](https://arxiv.org/abs/2605.27756) — POD mode를 전부 쓰는 대신 sparse하게 고르고 neural manifold reduction과 결합하는 MOR 작업입니다. CFD/thermal design loop에서 “빠르지만 해석 가능한 ROM”과 black-box surrogate 사이의 중간 지대를 만들 수 있습니다. `#MOR` `#POD` `#surrogate` `#dimensionality-reduction`
- [NPSolver: Neural Poisson Solver with Iterative Physics Supervision](https://arxiv.org/abs/2605.25786) · [code](https://github.com/intell-sci-comput/NPSolver) — irregular geometry와 mixed boundary condition에서 PCG refinement를 supervision signal로 쓰는 neural Poisson solver입니다. heat conduction, pressure projection, thermal boundary control처럼 Poisson류 subproblem이 많은 workflow에 실용적입니다. `#Poisson` `#heat-transfer` `#physics-supervision` `#irregular-domain`

### Thermal foundation models and regime-aware learning

- [Therm-FM: Foundation Model is ALL YOU NEED for 3D-ICs Thermal Simulation](https://arxiv.org/abs/2605.22663) · [code/model](https://github.com/haiyangxin/Therm-FM) — pretrained PDE foundation model을 3D-IC steady/transient thermal simulation에 adaptation합니다. electronics cooling/thermal design에서 high-fidelity FE data를 매번 새로 만들지 않고 cross-design reuse를 노리는 고신호 사례입니다. `#thermal` `#3D-IC` `#foundation-model` `#multi-fidelity`
- [NUCLEUS-MoE: Unified Model of Pool Boiling for Liquid Cooling](https://arxiv.org/abs/2605.27722) — pool boiling/liquid cooling을 mixture-of-experts로 다루는 thermal surrogate입니다. two-phase heat transfer는 regime이 바뀌면 물리가 확 달라지기 때문에, 단일 global model보다 regime-aware routing이 더 설득력 있습니다. `#boiling` `#liquid-cooling` `#MoE` `#heat-transfer`
- [Neural Operator-Based Surrogate Model for CFD: Helical Coil Steam Generator in Small Modular Reactor](https://arxiv.org/abs/2605.30277) — SMR helical coil steam generator transient CFD를 digital twin에 넣기 위한 ROM + neural operator surrogate입니다. 복잡 형상, transient thermal-hydraulics, operating envelope를 같이 보는 점이 장난감 benchmark와 다릅니다. `#SMR` `#thermal-hydraulics` `#CFD-surrogate` `#digital-twin`

### Engineering agents / executable representations

- [The Neural Compiler: Program-to-Network Translation for Hybrid Scientific Machine Learning](https://arxiv.org/abs/2605.22498) — first-order probabilistic programming language로 쓴 scientific model을 neural network로 컴파일하는 SciML 시스템입니다. PDE/physics prior를 손코딩 PyTorch로 매번 옮기는 부담을 줄이는 방향이라, hybrid SciML workflow의 interface design 관점에서 중요합니다. `#hybrid-SciML` `#program-to-network` `#physics-prior` `#tooling`
- [SchGen: PCB Schematic Generation with Semantic-Grounded Code Representations](https://arxiv.org/abs/2605.30345) — PCB schematic 생성 논문이지만 핵심은 CAD/CAE에도 통합니다: LLM에게 raw geometry syntax를 던지기보다 semantic operation, connectivity, constraint를 중간표현으로 주는 것이 correctness를 높입니다. `#design-automation` `#representation` `#LLM` `#engineering-agent`

## Best Find of the Day

**Operator Learning for Reconstructing Flow Fields from Sparse Measurements**를 오늘의 best find로 봅니다. VA가 보는 CFD surrogate의 실전 경로는 “CAD 하나 넣고 전체 field 예측”만이 아니라, sparse sensor/실험/운영 데이터와 결합해 현재 flow state를 복원하고 제어·설계 결정을 업데이트하는 쪽입니다. 이 방향은 thermal rig, wind/ventilation, vehicle aerodynamics, digital twin 모두에 재사용성이 큽니다.

## Possible Research Ideas

1. **Sparse-sensor thermal/flow digital twin mini-benchmark:** 2D heat conduction 또는 cavity flow에서 센서 3–10개만 주고 full field + control KPI를 복원하는 benchmark를 만든다.
2. **Surrogate readiness checklist:** inverse design, sparse reconstruction, thermal FM, MoE boiling을 묶어 surrogate를 “논문 성능”이 아니라 OOD split, sensor assimilation, uncertainty, operating envelope, UI/API로 점수화한다.
3. **Executable SciML interface sketch:** Neural Compiler 아이디어를 참고해 PDE prior, BC, material property, loss terms를 declarative spec으로 쓰고 PyTorch/JAX training code로 내려주는 작은 DSL을 설계한다.
4. **CAD/CAE representation note:** SchGen의 semantic primitive 관점을 FreeCAD/CadQuery/OpenSCAD agent에 적용해 “operation graph + constraints + validation hooks” 형태의 CAD-agent intermediate representation을 정리한다.
