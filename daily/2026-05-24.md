# Daily Research Radar — 2026-05-24

## Highlights

- **오늘의 핵심 신호는 thermal/CFD surrogate가 “범용 예측기”에서 solver·design·control loop 안의 재사용 가능한 부품으로 이동한다는 점입니다.** Therm-FM은 pretrained PDE foundation model을 3D-IC thermal simulation에 재사용하고, AOT-POT/Shodh-MoE 계열은 multi-PDE pretraining의 구조적 충돌을 줄이려 합니다.
- **thermal 쪽은 데이터 비용 절감이 가장 직접적인 가치입니다.** Therm-FM은 저비용 thermal-equivalent simulation + 소량 high-fidelity calibration을 묶어 cross-chip adaptation을 겨냥합니다. VA의 heat-transfer surrogate 관심사와 잘 맞습니다.
- **CFD surrogate는 geometry/mesh-aware, regime-aware inductive bias가 계속 중요해집니다.** ELGIN, shock-centered micro-nozzle, compositional NO는 큰 모델 하나보다 mesh coupling, coordinate registration, physics block composition을 강조합니다.
- **CAD/engineering agents는 생성보다 검증 루프가 관건입니다.** Physics-in-the-Loop, FEA-feedback CAD agent, CADTests/BenchCAD 흐름은 CAD artifact를 executable tests/FEA/physics tools로 닫힌 루프 안에서 평가해야 한다는 쪽으로 수렴합니다.
- **design automation은 sustainability/robotics까지 확장 중입니다.** fastener reduction 논문은 CAD graph + disassembly planning 결과를 설계 단순화 heatmap으로 되돌리는 예라, “해석 결과를 설계 변경 제안으로 번역”하는 좋은 패턴입니다.

## Added Resources

### Thermal, SciML & CFD surrogates

- [Therm-FM: Foundation Model is ALL YOU NEED for 3D-ICs Thermal Simulation](https://arxiv.org/abs/2605.22663) · [code/model repo](https://github.com/haiyangxin/Therm-FM) — pretrained PDE foundation model을 steady/transient 3D-IC thermal simulation에 adaptation하고, thermal-equivalent multi-fidelity training으로 high-fidelity data 요구량을 줄이는 논문. electronics cooling/heat-transfer surrogate에서 cross-design reuse를 논의할 때 고신호입니다. `#thermal` `#PDE-foundation-model` `#3D-IC` `#multi-fidelity`
- [AOT-POT: Adaptive Operator Transformation for Large-Scale PDE Pre-training](https://arxiv.org/abs/2605.15793) — 다양한 PDE operator를 그대로 한 모델에 밀어 넣기보다 adaptive operator transformation으로 더 정렬된 형태로 바꾸자는 접근. SciML foundation model의 “capacity scaling만으로는 부족하다”는 근거로 좋습니다. `#operator-learning` `#PDE-pretraining` `#foundation-model`
- [Eradicating Negative Transfer in Multi-Physics Foundation Models via Sparse Mixture-of-Experts Routing](https://arxiv.org/abs/2605.15179) — dense universal neural operator의 negative transfer를 sparse MoE routing으로 줄이려는 논문. open-channel flow와 porous media처럼 spectral/geometric 요구가 다른 regime을 한 모델에 넣을 때의 routing pattern이 중요합니다. `#multi-physics` `#MoE` `#neural-operator` `#negative-transfer`
- [Physics-informed GNN Surrogates for Turbulent Nanoparticle Dispersion](https://arxiv.org/abs/2605.19589) — OpenFOAM polyhedral mesh의 carrier flow와 Lagrangian particle motion을 함께 예측하는 ELGIN surrogate. indoor aerosol/particle-laden turbulent flow에서 mesh-aware surrogate와 transport metric이 왜 필요한지 보여줍니다. `#GNN` `#OpenFOAM` `#particle-dispersion` `#surrogate`
- [Physics Informed Neural Network-based Method for Accelerating Time-Periodic Unsteady CFD](https://arxiv.org/abs/2605.18340) — 긴 transient를 전부 적분하지 않고 one-period window에 직접 맞추는 PINN 기반 periodic CFD 접근. turbomachinery/oscillatory thermal-fluid 문제에서 “관심 없는 초기 transient 비용 제거”라는 실용 포인트가 있습니다. `#PINN` `#time-periodic` `#CFD` `#acceleration`

## Best Find of the Day

**Therm-FM**입니다. 이유는 단순히 “thermal surrogate가 나왔다”가 아니라, pretrained PDE foundation model + multi-fidelity adaptation + cross-chip few-shot calibration이라는 조합이 VA의 heat-transfer/SciML 관심사에 바로 연결되기 때문입니다. CFD/thermal surrogate 연구에서 매번 geometry/design마다 새 high-fidelity dataset을 크게 만드는 방식은 지속 가능하지 않습니다. Therm-FM은 transfer와 calibration을 중심에 둔 실용적 방향입니다.

## Possible Research Ideas

1. **Thermal foundation-model mini benchmark:** 2D/3D heat conduction toy cases에서 pretrained diffusion/PDE surrogate, FNO, classical ROM을 비교하고 target geometry sample 수를 5/10/30개로 제한해 adaptation curve를 본다.
2. **CAD-to-thermal artifact gate:** CadQuery/FreeCAD로 heat sink 또는 duct geometry를 만들고, executable CAD tests → meshability → thermal solver smoke test → surrogate prediction log까지 하나의 CI gate로 묶는다.
3. **Regime-aware surrogate study:** open-channel, porous, shock/thermal-front 데이터를 섞어 dense shared model vs MoE routing vs physics-block composition을 비교한다. metric은 평균 MSE뿐 아니라 regime별 failure와 conservation residual을 본다.
