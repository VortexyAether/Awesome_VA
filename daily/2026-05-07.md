# Daily Research Radar — 2026-05-07

## Highlights

- **Thermal/control 쪽 신호가 좋다.** 원자로 thermal-hydraulic forecasting, pouch-cell temperature estimation, electrothermal PINN처럼 “빠른 예측 + partial observability/control”로 바로 이어지는 논문들이 보였다.
- **CFD surrogate는 solver replacement보다 solver-in-the-loop / control-oriented reduced model 쪽이 실용적이다.** 오늘 추가한 thermal-hydraulic·battery·electrothermal 항목은 실제 센서/제어/운영 문제와 더 가깝다.
- **CAD/CAD automation 도구는 MCP·agent interface가 계속 늘고 있다.** FreeCAD robust MCP, CadQuery MCP, Text2CAD는 아직 early-stage지만 “GUI 대신 검증 가능한 API/tool surface”라는 방향성이 명확하다.
- **자율 실험/qualification은 engineering workflow의 다음 축이다.** Safe active learning 기반 sensor reliability paper는 CFD는 아니지만, thermal stress 조건에서 surrogate가 실험 선택과 안전 제약을 함께 다루는 좋은 패턴이다.

## Added Resources

### Thermal / control-oriented SciML

- [Graph Neural ODE Digital Twins for Control-Oriented Reactor Thermal-Hydraulic Forecasting Under Partial Observability](https://arxiv.org/abs/2604.07292) — advanced reactor supervisory control을 목표로 plant-wide thermal-hydraulic state를 빠르게 예측하는 graph neural ODE surrogate.
- [Physics-Informed Machine Learning for Pouch Cell Temperature Estimation](https://arxiv.org/abs/2604.14566) — indirect liquid cooling pouch cell의 온도 추정을 physics-informed ML로 다루는 짧지만 응용성이 명확한 battery thermal-management paper.
- [LSTM-PINN for Steady-State Electrothermal Transport](https://arxiv.org/abs/2604.14201) — 강하게 결합된 heat transfer, fluid flow, electric potential에서 multi-field consistency를 보존하려는 PINN 변형.
- [Autonomous Reliability Qualification of Ga2O3-based Hydrogen and Temperature Sensors via Safe Active Learning](https://arxiv.org/abs/2605.00868) — thermal/hydrogen stress 조건에서 Gaussian-process surrogate와 safe active learning으로 실험 qualification을 자동화하는 engineering-AI example.

### CAD, MCP & agentic engineering tools

- [FreeCAD Robust MCP server](https://github.com/spkane/freecad-addon-robust-mcp-server) — FreeCAD를 agent-callable CAD workbench/MCP bridge로 노출하려는 비교적 활발한 MCP implementation.
- [mcp-cadquery-server-public](https://github.com/mikekuniavsky/mcp-cadquery-server-public) — CadQuery code를 3MF/PNG/GLB artifact로 돌려주는 작은 MCP server; CAD generation 검증 루프에 맞는 형태.
- [Text2CAD](https://github.com/Toommo2/Text2CAD) — 자연어를 deterministic/validated CAD artifacts로 변환하려는 agent-driven workflow; 아직 초기지만 “validated engineering workflow”를 명시한다.

## Best Find of the Day

**Graph Neural ODE Digital Twins for Control-Oriented Reactor Thermal-Hydraulic Forecasting**가 오늘의 best pick. VA 관심사인 열전달/유체/제어가 한 번에 걸려 있고, 단순 field reconstruction이 아니라 partial observability 아래에서 supervisory control에 쓸 수 있는 surrogate를 지향한다. CFD surrogate 연구를 실제 제어 루프로 옮길 때 좋은 anchor paper가 될 수 있다.

## Possible Research Ideas

1. **Thermal-control surrogate benchmark:** reactor/pouch-cell/electrothermal 계열을 묶어 “sensor subset → full thermal state → control-relevant metrics” benchmark로 정리.
2. **CAD MCP validation loop:** FreeCAD/CadQuery MCP로 생성된 CAD를 STEP/STL export, meshing sanity check, 간단한 CFD/thermal case까지 자동 검증하는 최소 workflow 만들기.
3. **Safe active learning for CFD campaigns:** expensive CFD sweep에서 Gaussian-process/SafeOpt류로 실패 조건을 피하면서 informative design points를 고르는 실험 설계 pattern 검토.
