## Added Resources

### CFD / turbulence / surrogate validation

- [Spectrally Regularized Latent Flow Matching for Turbulence Generation](https://arxiv.org/abs/2606.11691) — latent diffusion/flow matching turbulence generator가 dissipation range amplitude를 과소표현하는 문제를 log-spectral objective로 직접 잡습니다. 256² DNS에서 deep-dissipation retained spectral power를 reconstruction 25%→94%, generation 20%→79%로 올렸다는 점이 핵심입니다. **Action: Read** `#turbulence` `#flow-matching` `#spectral-metrics` `#generative-surrogate`
- [Data-driven surrogate models for forecasting experimentally measured fluid flows](https://arxiv.org/abs/2606.10848) — PIV로 측정한 cylinder wake를 FCNN/U-Net/FNO/DMD 계열로 forecasting하며, noisy/incomplete experimental observations에서 transient/high-frequency energy가 무너지는 한계를 보여줍니다. 시뮬레이션 benchmark가 아니라 실험 계측 surrogate라 VA의 validation lens에 잘 맞습니다. **Action: Cite** `#experimental-fluid` `#PIV` `#surrogate-validation` `#FNO`
- [Robust Control of ECH Deposition Profiles on DIII-D](https://arxiv.org/abs/2606.13661) — TORBEAM surrogate neural network와 genetic optimizer로 tokamak ECH deposition profile을 실시간 최적화하고, gyrotron failure와 plasma parameter 변화에도 실험/ECE/offline ray tracing으로 검증합니다. CFD/thermal surrogate도 actuator failure와 operating shift를 control contract에 넣어야 한다는 좋은 사례입니다. **Action: Read** `#control` `#surrogate` `#robust-optimization` `#digital-twin`
- [Aerial Wildfire Suppression Planning with a Hybrid CNN-Cellular Automata Fire Model](https://arxiv.org/abs/2606.13633) — terrain/fuel/wind 기반 hybrid neural-cellular automata fire spread model과 gradient-based aerial drop planning을 결합하고 aleatoric/epistemic uncertainty로 suppression plan robustness를 봅니다. Thermal/fluid surrogate가 downstream intervention decision까지 연결될 때의 평가 구조로 참고할 만합니다. **Action: Watch** `#control` `#spatiotemporal-surrogate` `#uncertainty` `#planning`
- [Cepstral Analysis to accelerate Green-Kubo thermal conductivity calculations of Metal-Organic Frameworks](https://arxiv.org/abs/2606.13588) — machine-learned moment tensor potentials + Green-Kubo thermal conductivity 계산에서 cepstral analysis로 sampling time과 ad hoc parameter sensitivity를 줄입니다. Heat-transfer automation 쪽에서 “reproducible near-ab-initio thermal property pipeline”으로 쓸 수 있는 신호입니다. **Action: Save** `#thermal-conductivity` `#materials` `#ML-potential` `#automation`

### CAD / engineering agents / evaluation harnesses

- [IterCAD: An Iterative Multimodal Agent for Visually-Grounded CAD Generation and Editing](https://arxiv.org/abs/2606.13368) — drawing-to-code, text-to-code, interactive editing을 executable CAD sandbox와 multi-turn loop로 묶고, IterCAD-Bench와 CD-TR/AUC-TR metric을 제안합니다. CAD agent 평가에서 “valid code만 살아남은 survivor bias”를 줄이려는 점이 고신호입니다. **Action: Test** `#CAD` `#multimodal-agent` `#CAD-benchmark` `#geometry-validation`
- [ComAct: Reframing Professional Software Manipulation via COM-as-Action Paradigm](https://arxiv.org/abs/2606.13239) — GUI computer-use 대신 Windows COM을 deterministic action/program synthesis interface로 보고, real industrial CAD software용 ComCADBench/ComActor를 제안합니다. 상용 CAD/CAE agent는 screenshot control보다 stable executable action layer가 필요하다는 강한 신호입니다. **Action: Read** `#CAD-agent` `#COM` `#professional-software` `#benchmark`
- [EurekAgent: Agent Environment Engineering is All You Need For Autonomous Scientific Discovery](https://arxiv.org/abs/2606.13662) — autonomous scientific discovery의 병목을 agent workflow가 아니라 permissions, artifacts, budget, human-in-the-loop이 포함된 environment engineering으로 정의합니다. VA의 CFD/CAD agent 실험도 권한·artifact·budget·Git collaboration을 먼저 설계해야 한다는 운영 프레임입니다. **Action: Save** `#scientific-agents` `#environment-engineering` `#artifact-management` `#governance`
- [AgentRivet: an automated system for producing Rivet routines from journal publications](https://arxiv.org/abs/2606.13535) — HEP 논문에서 Rivet C++ analysis routines를 LLM multi-step workflow로 생성하고 code/physics review를 넣습니다. 분야는 다르지만 “논문→검증 가능한 실행 artifact” 자동화의 좋은 패턴이며 CFD benchmark reproduction agent에 전이 가능성이 있습니다. **Action: Watch** `#research-agent` `#artifact-generation` `#quality-control` `#reproducibility`

## Best Find of the Day

**Data-driven surrogate models for forecasting experimentally measured fluid flows** 를 오늘의 best find로 봅니다. 이유는 냉정합니다. 대부분 CFD-AI surrogate는 깨끗한 simulation data에서 잘 되는 그림을 보여주지만, 실제 engineering loop는 noisy PIV/sensor, incomplete state, transient features, high-frequency energy loss와 싸웁니다. 이 논문은 faster-than-real-time 가능성과 동시에 “실험 데이터 surrogate가 어디서 무너지는지”를 같이 보여줘서 VA의 validation/evaluation agenda에 바로 들어갑니다.

## Suggested Experiments / Follow-up

1. **Experimental-flow surrogate stress test:** PIV/particle-image류 open dataset에서 FNO/U-Net/DMD를 비교하되 RMSE 외에 spectrum, vortex shedding frequency, phase drift, high-frequency energy retention을 별도 metric으로 둔다.
2. **CAD agent benchmark smoke test:** IterCAD/ComAct 기준을 빌려 FreeCAD/CadQuery agent 결과를 `code executes?`, `geometry valid?`, `dimensions match?`, `STEP export?`, `meshable?`로 나누어 평가한다.
3. **Surrogate control contract:** DIII-D ECH 사례처럼 actuator failure, model latency, operating shift, fallback solver/offline verifier를 CFD/thermal control surrogate checklist에 넣는다.
4. **Paper-to-artifact agent pattern:** AgentRivet 구조를 CFD benchmark reproduction에 맞게 “paper extraction → case setup → solver run → validation report → human review”로 재해석한다.
