## Added Resources

### CFD / flow control / thermal-fluid surrogate

- [Drag reduction or reward hacking? Recurrent multi-agent reinforcement learning that earns its reward](https://arxiv.org/abs/2606.06227) — wall turbulence drag reduction에서 reward가 실제 에너지 절감과 어긋나면 controller가 “보상만 잘 받는” 퇴행 전략을 찾는다는 점을 짚고, differentiable projection·recurrent policy·true wall-power reward로 수정합니다. Flow-control RL에서 offline success figure보다 accounting과 closed-loop physics가 더 중요하다는 좋은 경고입니다. `#flow-control` `#turbulence` `#reinforcement-learning` `#reward-hacking`
- [Reconstructing Unobservable Temperature Fields via Simulation-Aided Intelligent Sensing](https://arxiv.org/abs/2606.04582) — randomized physics-based simulation으로 synthetic dataset을 만들고 sparse embedded sensors에서 내부 temperature field를 실시간 복원합니다. Thermal digital twin에서 “센서를 못 넣는 위치의 상태를 어떻게 안전하게 추정할 것인가”라는 실무 문제에 직접 닿습니다. `#heat-transfer` `#thermal-sensing` `#digital-twin` `#simulation-data`
- [Validation-Gated Multi-Agent Governance for Online Adaptation of Thermal-Hydraulic Surrogate Models under Operating-Regime Shift](https://arxiv.org/abs/2606.03321) — Monitor/Diagnosis/Adaptation/Safety-Auditor/Orchestrator agent가 thermal-hydraulic surrogate를 감시하되, 최종 교체는 deterministic champion-challenger validation gate가 담당합니다. Agentic engineering에서 agent는 “결정자”보다 auditable reviewer여야 한다는 패턴이 좋습니다. `#thermal-hydraulic` `#surrogate` `#multi-agent` `#validation-gate`
- [3D Underwater Path Planning via Generative Flow Field Surrogates](https://arxiv.org/abs/2606.06077) — RANS propeller-wake surrogate를 AUV launch/recovery path planning에 연결합니다. CFD surrogate 평가가 field RMSE에서 끝나지 않고 downstream planning objective까지 가야 한다는 applied example입니다. `#CFD-surrogate` `#RANS` `#path-planning` `#control`
- [A Surrogate Model for Proton Spectrum Prediction to Map Transitions in Laser-Ion Acceleration](https://arxiv.org/abs/2606.06210) — β-VAE + MLP dual-branch surrogate와 uncertainty ensemble로 laser-ion acceleration spectrum과 regime transition을 예측합니다. CFD는 아니지만 high-dimensional physics output surrogate + UQ + closed-loop facility control이라는 pattern이 engineering surrogate 설계에 참고됩니다. `#physics-surrogate` `#UQ` `#closed-loop-control` `#regime-transition`

### CAD / design automation / decision layer

- [Bridging CAD and Data-Driven Design: Attributed Feature Graphs for Engineering Design](https://arxiv.org/abs/2606.06405) — CAD feature를 node/relation/attribute graph로 표현해 data-driven design과 CAE surrogate에 필요한 design intent를 보존하려는 접근입니다. CAD-to-CAE 자동화에서 mesh/voxel만 보는 모델의 약점을 보완할 수 있습니다. `#CAD` `#feature-graph` `#design-automation` `#CAE`
- [BRepCLIP: Contrastive Multimodal Pretraining on BRep Primitives for CAD Understanding](https://arxiv.org/abs/2606.05515) — native B-Rep face/edge primitive를 language/image embedding과 정렬합니다. CAD agent가 “그럴듯한 형상”을 넘어서 boundary semantics와 feature meaning을 다루려면 이런 B-Rep-level representation이 중요합니다. `#CAD` `#BRep` `#contrastive-learning` `#geometry`
- [From data to decisions: Bayesian modelling and global sensitivity analysis for flotation control](https://arxiv.org/abs/2606.06173) — GP surrogate, Sobol sensitivity, SHAP을 결합해 flotation air recovery 제어 변수의 의사결정 근거를 만듭니다. Industrial process control에서 black-box predictor보다 sensitivity + uncertainty + operator-readable explanation 조합이 실용적입니다. `#process-control` `#Bayesian-modeling` `#sensitivity-analysis` `#decision-support`

## Best Find of the Day

**Drag reduction or reward hacking?** 를 오늘의 best find로 봅니다. 이유는 VA가 관심 있는 control/agentic engineering workflow에서 가장 위험한 실패가 “모델이 틀렸다”보다 “평가지표가 설계 의도를 배신했다”이기 때문입니다. CFD/thermal/control surrogate도 앞으로 field error, reward, objective, safety gate를 따로 검증해야 합니다.

## Possible Research Ideas

1. **Honest-control benchmark:** flow/thermal control surrogate에서 reward, 실제 에너지/압력손실/열안전 margin, actuator cost를 분리해 기록하는 benchmark sheet를 만든다.
2. **Sparse thermal sensing mini-project:** randomized simulation dataset으로 sparse sensor → full temperature field reconstruction을 재현하고, Kriging/POD-NN/FNO baseline을 비교한다.
3. **Agentic surrogate governance spec:** thermal-hydraulic gate 논문처럼 Monitor, Diagnoser, Adapter, Auditor, Orchestrator 역할을 정의하되 최종 model swap은 deterministic validation contract에 맡긴다.
4. **CAD feature graph audit:** Attributed Feature Graphs/BRepCLIP을 기준으로 CAD-to-CAE pipeline이 feature semantics, named boundaries, material/BC metadata를 보존하는지 checklist를 만든다.
5. **Decision-level surrogate metric:** AUV path planning 사례처럼 CFD surrogate를 field RMSE뿐 아니라 downstream planner/controller objective error로 평가한다.
