# Daily Research Radar — 2026-05-12

## Highlights

- **CAD agent 평가가 드디어 “예쁜 결과”에서 executable tests로 이동 중이다.** CADTestBench/CADTests는 text-to-CAD 결과물을 CadQuery 기반 테스트로 검증해, prompt의 치수·위상 요구사항을 직접 확인한다.
- **CAD MCP 생태계는 FreeCAD/build123d/SolidWorks로 빠르게 갈라지고 있다.** 핵심 차이는 GUI 제어가 아니라 measurement, export, snapshot, validation 같은 agent-readable operation surface다.
- **Thermal SciML은 active learning + interpretable surrogate 조합이 실용적이다.** Thermal Energy System digital twin 논문은 GRU 성능뿐 아니라 SINDyC/MvG-SINDyC의 해석성·불확실성·데이터 효율을 같이 비교한다.
- **모든 CFD surrogate가 neural operator일 필요는 없다.** Visco-plastic shallow flow ROM은 구조화된 parameter space와 복잡한 rheology에서는 tensor/ROM 계열도 여전히 강한 baseline이라는 점을 상기시킨다.

## Added Resources

### CAD, CAD automation & MCP

- [Text-to-CAD Evaluation with CADTests](https://arxiv.org/abs/2605.07807) · [code](https://github.com/dimitrismallis/CADTestBench) · [dataset](https://huggingface.co/datasets/dimitrismallis/CADTestBench) — CadQuery 기반 executable tests로 생성 CAD가 prompt의 치수/위상 요구를 만족하는지 검증. CAD agent 평가에 바로 써먹을 수 있는 고신호 benchmark. `#CAD` `#evaluation` `#CadQuery`
- [build123d-mcp](https://github.com/pzfreo/build123d-mcp) — build123d CAD 작업을 MCP tool로 노출. execute/render/measure/clearance/cross-section/export/snapshot/shape_compare가 있어 “blind script generation”을 줄이는 방향. `#MCP` `#build123d` `#CAD-automation`
- [neka-nat/freecad-mcp](https://github.com/neka-nat/freecad-mcp) — FreeCAD workbench/addon + RPC/MCP server. text-only feedback, remote control 설정, 2D drawing → part demo 등 agent-CAD 연결의 실사용 패턴을 보여준다. `#FreeCAD` `#MCP` `#CAD-agent`
- [SolidworksMCP-TS](https://github.com/vespo92/SolidworksMCP-TS) — SolidWorks COM 자동화를 TypeScript MCP server로 감싼 alpha project. direct COM과 VBA macro fallback을 섞는 설계가 흥미롭다. `#SolidWorks` `#MCP` `#COM`

### Thermal/control SciML & flow surrogates

- [Physics-based Digital Twins for Integrated Thermal Energy Systems Using Active Learning](https://arxiv.org/abs/2605.06756) — Modelica thermal-system simulation에 SINDyC/MvG-SINDyC/FNN/GRU surrogate와 active learning을 붙여 glycol heat exchanger control surrogate를 만든다. `#thermal` `#digital-twin` `#active-learning`
- [Reduced-Order Modeling of Parameterized Visco-Plastic Shallow Flows](https://arxiv.org/abs/2605.06526) — Herschel-Bulkley shallow flow의 moving front/yield surface를 non-intrusive ROM으로 다룸. neural operator baseline과 비교할 고전적이지만 강한 surrogate 축. `#ROM` `#non-Newtonian` `#surrogate`

## Best Find of the Day

**CADTestBench / CADTests**. CAD-to-CAE agent를 진짜 engineering workflow에 넣으려면 “모양이 그럴듯함”보다 치수, feature, topology, export, meshability가 통과됐는지가 중요하다. 이 논문은 CAD generation을 code synthesis처럼 보고 executable tests로 평가하므로, VA의 CAD 자동화/CAE handoff 체크리스트에 바로 옮겨올 수 있다.

## Possible Research Ideas

1. **VA-CADTests mini benchmark:** bracket, duct, heat sink, manifold 같은 5개 geometry prompt에 대해 build123d/CadQuery 테스트를 만들고, FreeCAD/build123d MCP agent 출력물을 자동 채점한다.

3. **CAD MCP bake-off:** build123d-mcp, FreeCAD MCP, SolidWorks MCP를 같은 part spec으로 비교하고 `geometry_tests.json`, `export_manifest.json`, `meshability_report.md`를 공통 artifact로 강제한다.
