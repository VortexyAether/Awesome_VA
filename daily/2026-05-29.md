# Daily Research Radar — 2026-05-29

## Highlights

- **오늘의 sciML 신호는 “엔지니어링 surrogate의 배치/전이/GUI화”입니다.** 자동차 공력 transfer learning, CFDTwin, coronary hemodynamics RBF decoder가 모두 “빠른 field prediction”을 실제 설계·디지털트윈 workflow에 어떻게 넣을지 묻고 있습니다.
- **열/냉각 쪽은 데이터와 물리 단위가 같이 중요해지고 있습니다.** lunar rover thermal profiling과 pool boiling/liquid-cooling MoE는 thermal design surrogate가 단순 regression보다 radiative/phase-change 조건, operating envelope, uncertainty를 함께 다뤄야 함을 보여줍니다.
- **CFD surrogate의 신뢰성 축은 posterior, control, closure로 확장 중입니다.** Bayesian inverse design, CINOC PDE control, algebraic Reynolds-stress closure는 “예측이 맞는가”를 넘어 posterior 보존, actuator/sensor cardinality 일반화, RANS solver 내부 분포 이동을 봅니다.
- **CAD/agentic design은 benchmark가 더 엄격해지는 방향입니다.** MUSE는 text-to-CAD를 geometry similarity가 아니라 manufacturability, functionality, assemblability로 평가하자고 제안해 VA의 CAD→CAE 자동화 관심사와 잘 맞습니다.
- **실무 관점의 best signal은 CFDTwin + MUSE 조합입니다.** 하나는 Fluent 결과를 POD-NN surrogate GUI/Python toolkit로 묶고, 다른 하나는 생성 CAD의 산업적 유효성을 평가합니다. CAD 생성과 CFD surrogate를 이어 붙이려면 둘 사이에 validation contract가 필요합니다.

## Added Resources

### CFD / thermal-fluid surrogates & digital twins

- [CFDTwin: An open-source GUI and Python toolkit for POD-NN surrogate modeling of ANSYS Fluent simulations](https://arxiv.org/abs/2605.27725) — Fluent 기반 thermal-fluid CFD 결과를 POD-NN surrogate로 만들고 GUI/Python toolkit 형태로 제공하려는 논문입니다. 반복 CFD solve가 비싼 설계 최적화·불확실성 분석·digital twin workflow에서 “논문 모델”을 엔지니어가 다루는 툴로 내리는 신호라 중요합니다. 그룹/소속: 초록 메타에 명시 없음. `#CFD` `#surrogate` `#Fluent` `#digital-twin`
- [NUCLEUS-MoE: Unified Model of Pool Boiling for Liquid Cooling](https://arxiv.org/abs/2605.27722) — pool boiling/liquid cooling을 mixture-of-experts로 통합 모델링하려는 thermal surrogate입니다. two-phase boiling은 phase change, turbulence, material/fluid condition이 얽혀 단일 global regressor가 흔들리기 쉬운데, regime-aware expert routing이 liquid-cooling AI 모델의 현실적 방향입니다. 그룹/소속: 초록 메타에 명시 없음. `#heat-transfer` `#liquid-cooling` `#boiling` `#MoE`
- [Faster Thermal Profiling of a Lunar Rover with Machine Learning Adapted Finite Difference Model](https://arxiv.org/abs/2605.27651) — lunar rover의 radiative/환경 조건이 강한 thermal model을 ML-adapted finite-difference 방식으로 빠르게 만들려는 작업입니다. 우주 로버라는 특수 도메인이지만 “물리 grid model + ML correction” 패턴은 배터리/전자장비/냉각 구조의 빠른 thermal profiling에도 참고할 만합니다. 그룹/소속: 초록 메타에 명시 없음. `#thermal` `#finite-difference` `#space-systems` `#digital-twin`
- [From Centerlines to Hemodynamics: Anisotropic RBF Decoders for Coronary Arteries](https://arxiv.org/abs/2605.27578) — coronary artery centerline에서 pressure/wall-shear-stress 같은 hemodynamic metric을 빠르게 추정하는 anisotropic RBF decoder입니다. 의료 CFD지만 irregular vascular geometry → field metric surrogate라는 점에서 pipe/duct/network형 CFD surrogate와 연결됩니다. 그룹/소속: Reza Akbarian Bafghi, Sukirt Thakur, Maziar Raissi. `#CFD` `#geometry-to-field` `#RBF` `#hemodynamics`
- [Adapting Automotive Aerodynamics Surrogates to New Vehicle Families via Transfer Learning](https://arxiv.org/abs/2605.27968) — automotive CFD surrogate가 새 vehicle family로 넘어갈 때 geometry encoder representation이 얼마나 transfer되는지 leave-one-family-out으로 검증합니다. 산업 CFD surrogate의 병목은 한 dataset accuracy보다 family/OOD 전이이므로, VA가 보는 CAD/CFD design loop에 바로 관련 있습니다. 그룹/소속: Seunghwan Keum, Alok Warey. `#aerodynamics` `#transfer-learning` `#CFD-surrogate` `#automotive`

### Control, closure & optimization

- [CINOC: Cardinality-Invariant Neural Operator Policies for Scalable PDE Control](https://arxiv.org/abs/2605.25867) — sensor/actuator/agent 개수가 바뀌어도 동작하는 cardinality-invariant neural operator policy를 제안합니다. thermal-fluid control에서 actuator layout이나 sensor count가 바뀔 때 policy를 다시 학습하지 않는 방향이라, SciML surrogate를 control loop에 넣는 문제와 맞닿아 있습니다. 그룹/소속: 초록 메타에 명시 없음. `#PDE-control` `#neural-operator` `#scalable-control`
- [Deep Learning-based Algebraic Reynolds Stress Closures for RANS Simulations of Turbulent Flows](https://arxiv.org/abs/2605.26358) — offline-trained ML closure가 RANS solver 안에서 distribution shift를 겪는 문제를 다루는 algebraic Reynolds-stress closure 작업입니다. turbulence ML의 실제 병목은 model form보다 solver-coupled stability와 closure consistency인 경우가 많아, CFD-AI 과장 필터로 좋습니다. 그룹/소속: Daniel Dehtyriov, Jonathan F. MacArt, Justin Sirignano. `#RANS` `#turbulence` `#closure-model` `#CFD`
- [PIRS: Physics-Informed Reward Shaping for SAC-Based Building Energy Management](https://arxiv.org/abs/2605.28232) — building energy management에서 comfort/efficiency tradeoff reward를 physics-informed shaping으로 설계하는 SAC 기반 control 논문입니다. 직접 CFD는 아니지만 thermal control agent가 reward hacking을 피하려면 물리 proxy와 comfort constraints가 필요하다는 점에서 유용합니다. 그룹/소속: 초록 메타에 명시 없음. `#thermal-control` `#RL` `#building-energy` `#physics-informed`

### CAD / design automation benchmark

- [MUSE: Benchmarking Manufacturable, Functional, and Assemblable Text-to-CAD Generation](https://arxiv.org/abs/2605.28579) — text-to-CAD를 단일 part geometric similarity가 아니라 manufacturability, functionality, assemblability까지 평가하자는 benchmark입니다. CAD automation이 CAE/제조까지 이어지려면 “그럴듯한 형상”보다 조립 가능성, 기능, 제조 제약이 평가되어야 하므로 weekly synthesis 후보입니다. 그룹/소속: 초록 메타에 명시 없음. `#CAD` `#benchmark` `#text-to-CAD` `#manufacturability`

## Possible Research Ideas

1. **Fluent/OpenFOAM surrogate asset template:** CFDTwin을 참고해 case metadata, geometry parameter, POD basis, NN weights, validation plots, operating envelope를 하나의 artifact contract로 묶는 템플릿을 만든다.
2. **Thermal cooling regime router:** NUCLEUS-MoE식으로 single-phase / nucleate boiling / transition / CHF 근처 데이터를 나눠 expert routing surrogate를 만들고, uncertainty가 높은 regime에서 classical solver fallback을 건다.
3. **CAD benchmark → CAE benchmark bridge:** MUSE의 manufacturable/functional/assemblable 평가 항목에 meshability, boundary-condition assignability, solver-run success, KPI validity를 추가한 CAD-to-CFD benchmark를 설계한다.
4. **OOD vehicle-family transfer audit:** automotive surrogate에서 family holdout, geometry encoder freezing, few-shot fine-tuning, wake/pressure coefficient error를 비교해 “새 형상군에 대한 전이 가능성”을 정량화한다.
5. **Control-ready surrogate test:** CINOC/PIRS 관점으로 sensor/actuator 개수 변화, physical reward shaping, rollout safety constraint를 thermal-fluid control toy problem에 넣어본다.
