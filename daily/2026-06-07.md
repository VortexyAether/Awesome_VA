## Added Resources

### SciML / CFD / thermal-fluid surrogates

- [Adapting Automotive Aerodynamics Surrogates to New Vehicle Families via Transfer Learning](https://arxiv.org/abs/2605.27968) — 411개 external-aero CFD case로 pretrain한 Transformer surrogate를 unseen vehicle family에 적응시키는 문제를 다룹니다. CAD/geometry encoder가 family shift를 견디는지 검증한다는 점에서 industrial CFD surrogate에 바로 연결됩니다. `#CFD-surrogate` `#automotive-aero` `#transfer-learning` `#geometry-encoder`
- [Accelerating Bayesian inverse design in computational fluid dynamics using neural operators](https://arxiv.org/abs/2605.26059) — shock-dominated CFD inverse design에서 neural operator surrogate가 MCMC/posterior geometry uncertainty에 어떤 영향을 주는지 봅니다. 단순 속도 향상이 아니라 inverse-design posterior를 망치지 않는지가 핵심이라 설계 자동화 관점에서 중요합니다. `#CFD` `#inverse-design` `#neural-operator` `#uncertainty`
- [Neural Operator-Based Surrogate Model for CFD: Helical Coil Steam Generator in Small Modular Reactor](https://arxiv.org/abs/2605.30277) — SMR helical-coil steam generator transient CFD를 neural operator로 대체해 digital twin 실시간 열수력 예측을 노립니다. thermal-hydraulic geometry-specific surrogate의 좋은 추적 대상입니다. `#thermal-hydraulic` `#SMR` `#digital-twin` `#neural-operator`
- [Faster Thermal Profiling of a Lunar Rover with Machine Learning Adapted Finite Difference Model](https://arxiv.org/abs/2605.27651) — lunar rover의 radiation/large-gradient thermal model을 ML-adapted finite-difference로 빠르게 돌리는 접근입니다. onboard autonomy와 mission design에서 “물리 모델을 버리지 않고 빠르게 보정”하는 패턴이 유용합니다. `#heat-transfer` `#thermal-modeling` `#finite-difference` `#autonomy`
- [Learned Response-Field Inertia Operator for HEC-RAS 2D Water-Surface Elevation Prediction](https://arxiv.org/abs/2606.06385) — HEC-RAS 2D native nonuniform cell 위에서 water-surface elevation surrogate를 평가합니다. raster remapping 없이 solver-consistent cell representation을 유지하는 점이 CFD surrogate evaluation에 좋은 힌트입니다. `#hydrodynamics` `#surrogate` `#HEC-RAS` `#nonuniform-mesh`
- [Learning and Inferring Multiphase Flow Dynamics in Porous Media using Scientific Machine Learning](https://arxiv.org/abs/2606.05448) — FluidFlower CO2 injection experiment에서 surrogate modeling과 Bayesian inference를 결합합니다. porous multiphase flow처럼 nonlinear/limited-observation 문제가 강한 영역에서 SciML inverse workflow를 볼 수 있습니다. `#multiphase-flow` `#porous-media` `#Bayesian-inference` `#SciML`

### Foundation models / turbulence / control

- [Emergent Transfer of a Physics Foundation Model from Simulation to Laboratory Turbulence](https://arxiv.org/abs/2606.01470) — Rayleigh-Taylor instability에서 simulation-trained physics foundation model의 lab experiment transfer를 검증합니다. CFD-AI에서 “benchmark score”보다 sim-to-lab transfer가 더 중요한 평가축이 되고 있다는 신호입니다. `#physics-foundation-model` `#turbulence` `#sim-to-lab` `#RTI`
- [Uncovering Extreme Event Mechanisms for Prediction and Control with Sensitivity-Balanced Projections](https://arxiv.org/abs/2606.05618) — chaotic dynamical systems의 extreme event mechanism을 adjoint snapshot 기반 projection으로 해석하고 suppression controller까지 연결합니다. flow/control 쪽에서는 black-box predictor보다 interpretable reduced mechanism + controller 패턴이 더 실용적입니다. `#control` `#dynamical-systems` `#extreme-events` `#reduced-order-modeling`

### CAD / engineering automation / agent protocols

- [BRepCLIP: Contrastive Multimodal Pretraining on BRep Primitives for CAD Understanding](https://arxiv.org/abs/2606.05515) — native CAD B-Rep face/edge primitive를 language/image embedding과 정렬합니다. CAE handoff에는 mesh-only representation보다 B-Rep/feature-level semantics가 더 쓸모 있을 가능성이 큽니다. `#CAD` `#BRep` `#contrastive-learning` `#geometry`
- [ghbalf/freecad-ai](https://github.com/ghbalf/freecad-ai) — natural language로 FreeCAD 모델을 생성하는 AI workbench입니다. MCP 서버는 아니지만 FreeCAD 내부 workbench형 CAD agent UX 참고로 볼 만합니다. `#FreeCAD` `#CAD-agent` `#text-to-CAD`
- [jdilla1277/agentcad](https://github.com/jdilla1277/agentcad) — AI agent용 CAD CLI와 MCP server를 제공하는 초기 프로젝트입니다. CadQuery/agent-facing CAD interface의 작은 구현 신호로 추적하되, 실사용 전 geometry validation과 export 안정성 검증이 필요합니다. `#MCP` `#CAD` `#CadQuery` `#agentic-engineering`

## Best Find of the Day

**Adapting Automotive Aerodynamics Surrogates to New Vehicle Families via Transfer Learning**를 오늘의 best find로 봅니다. 이유는 CFD surrogate의 진짜 병목이 “한 데이터셋에서의 낮은 error”가 아니라 새로운 geometry family로 넘어갈 때 representation이 살아남는지이기 때문입니다. VA가 CAD/CFD surrogate를 생각할 때도 geometry encoder transfer, family split, downstream drag/pressure objective error를 같이 봐야 합니다.

## Possible Research Ideas

1. **Family-shift CFD surrogate protocol:** 같은 차/덕트/열교환기 family 내부 split과 unseen family split을 분리해서 geometry encoder transfer를 평가한다.
2. **Thermal digital-twin safety gate:** SMR/lunar rover 사례를 참고해 thermal surrogate에 regime detector, uncertainty flag, physics residual check, rollback rule을 붙이는 mini spec을 만든다.
3. **Solver-native cell benchmark:** HEC-RAS 논문처럼 OpenFOAM/polyMesh native cell graph에서 field surrogate를 평가하고, raster/voxel remap error를 따로 기록한다.
4. **CAD-to-CAE representation audit:** BRepCLIP/FreeCAD MCP류를 대상으로 B-Rep face semantics, named boundaries, STEP/STL export, meshability를 테스트하는 checklist를 만든다.
5. **Sim-to-lab turbulence reading list:** RTI foundation model transfer를 시작점으로 simulation → experiment transfer 논문을 따로 모아 CFD-AI reliability 섹션을 만든다.
